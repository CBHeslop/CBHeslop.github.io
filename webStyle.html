Web Style Layout

This project was meant to demonstrate a web page layout focusing on typography and how to visually discern the different parts of a web page. I started with separating everything into paragraph styles using different colors with InDesign. I then sketched different layouts that I thought would look good using design principles. After deciding on a layout and adjusting the typography, I created a header for the website called Insights. Once deciding on the layout for the header and deciding to strictly stay with just typography and no extra logos, this is the direction of the website. I had the chance to adjust many things as I received countless feedback from other designers.
